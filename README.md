# popup-message
A simple popup success message design built using HTML, CSS, and JavaScript. Clicking the button shows a smooth animated popup with a thank-you message.
# Popup Success Message | Era21Tech


## Features
- Smooth scale animation popup
- Clean UI with centered layout
- Easy to customize text, image, and colors
- Works on all modern browsers

## Files
- `index.html` — Main popup structure and JS 
- `style.css` — Styling and popup animation 

## How to Use
1. Download the project.
2. Open `index.html` in any browser.
3. Click the **Submit** button to open the popup.
4. Press **OK** to close it.

## Customization
- Replace the success icon in `/images/`
- Update popup text in `index.html`
- Modify colors and animations in `style.css`
